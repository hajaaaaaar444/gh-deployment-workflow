# GitHub Pages Deployment

Description: A project to learn the notion of continuous integration and continuous deployment. A GitHub action to deploy a static website to GitHub Pages.

## How does it work

Every push to the main branch that changes the `index.html` file triggers the workflow to run and deploy the website to GitHub Pages.

## Technologies used

- GitHub Actions
- GitHub Pages
- HTML
